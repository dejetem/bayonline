# Installation
## Prerequisites:
- Node
- npm
- MongoDB Atlas URI


## Setup:
- `clone the repository`
- `update MongoDB Atlas URI in .env`
- `update PORT in .env`
- `update JWT_SECRET in .env`
- `run npm install`
- `run npm run dev`

## Api
- Base `http://localhost:5000/graphql`
- `Apollo Studio work best on google chrome browser`
